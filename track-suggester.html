There are many programming languages that have different advantages over others depending on what you want to do with them. With so many options, it can be hard to figure out which one to choose. Click on the track options below for more information about each one.

Epicodus Tracks

Ruby/Rails

Ruby is the language for interactive web applications, involved with doing things like creating accounts, taking in information, and working with dynamic content.

PHP/Drupal

PHP is the language used for content-heavy websites with little user interaction, great for content-management when building out websites for clients.

Java/Android

Java is a language used for building a wide variety of applications, including building Android user interfaces.

CSS/Design

CSS/Design focuses on the visual look and design of a webpage, otherwise known as front-end. This track is great for people interested in the look and feel of a webpage.

C#/.NET

C# is a well-established language mostly used for building internal software, particularly for business software for larger companies.
