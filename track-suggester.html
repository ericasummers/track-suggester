Which Epicodus track should I take?

There are many programming languages that have different advantages over others depending on what you want to do with them. With so many options, it can be hard to figure out which one to choose. Epicodus offers five different main tracks:

Epicodus Tracks

  1. Ruby/Rails
  2. PHP/Drupal
  3. Java/Android
  4. CSS/Design
  5. C#/.NET

For help figuring out which one might be right for you, please take the following survey. The result at the bottom will show a suggestion based on your responses of which track might best suit you.
